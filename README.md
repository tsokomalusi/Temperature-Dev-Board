# Temperature-Dev-Board

STM32-based temperature monitoring board featuring:

 10k NTC thermistor
- ADC temperature measurement
- OLED display
- USB power
- Expansion headers

## Hardware

- MCU: STM32
- Sensor: 10k NTC Thermistor
- Display: OLED
-
